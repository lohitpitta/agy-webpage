# AGY Webpage

A simple Flask-based web application created with the help of Antigravity CLI (AGY).

## Features

- Flask backend
- HTML templates
- Static CSS styling
- Lightweight and easy to run
- Beginner-friendly project structure

## Project Structure

```text
project/
├── app.py
├── requirements.txt
├── templates/
├── static/
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/lohitpitta/agy-webpage.git
cd agy-webpage
```

2. Create a virtual environment

```bash
python -m venv venv
```

3. Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

4. Install dependencies

```bash
pip install -r requirements.txt
```

5. Run the application

```bash
python app.py
```
